학생 성적 리포트

리스트 · 딕셔너리 · 반복문 · 선택문 · 함수를 모두 활용하는 종합 예제

list dict for 반복문 if / elif / else 함수 (def) f-string

1. 학습 목표

📋
데이터 구조 활용
학생 데이터를 리스트와 딕셔너리로 구성하는 방법을 이해한다.
🔁
반복 처리
for 반복문으로 여러 학생을 순서대로 처리하는 패턴을 익힌다.
🔀
조건 분기
if / elif / else 로 등급·합격 여부를 판단하는 로직을 작성한다.
⚙️
함수 분리
기능 단위로 함수를 나눠 코드의 재사용성과 가독성을 높인다.

2. 전체 프로그램 구조

이 예제는 3개의 함수와 1개의 데이터 변수로 이루어져 있습니다.

1
students 데이터 딕셔너리를 담은 리스트 – 이름과 점수 목록 저장
2
calculate_average(score_list) 점수 목록을 받아 평균을 계산해 반환
3
make_level(avg) 평균 점수를 받아 등급 문자(A/B/C/D)를 반환
4
print_report(student_list) 전체 학생을 순회하며 위 두 함수를 호출해 결과 출력

3. 데이터 구조 이해

학생 정보를 저장하는 students 변수를 살펴봅시다.

데이터 정의 
students = [
    {"name": "민지", "scores": [88, 92, 81]},
    {"name": "준호", "scores": [70, 65, 72]},
    {"name": "서연", "scores": [95, 90, 98]},
]
💡
중첩 자료구조
바깥쪽 [ ]은 리스트(학생 목록), 안쪽 { }는 딕셔너리(한 명의 정보), 딕셔너리 안의 "scores" 값도 다시 리스트입니다. 이렇게 자료구조를 중첩해서 표현하면 복잡한 데이터도 깔끔하게 담을 수 있습니다.

접근 방법

인덱스 & 키 접근 
# 첫 번째 학생 딕셔너리 전체
students[0]            # {"name": "민지", "scores": [88, 92, 81]}

# 첫 번째 학생 이름
students[0]["name"]    # "민지"

# 첫 번째 학생 두 번째 점수
students[0]["scores"][1]  # 92

4. 함수별 코드 해설

① calculate_average – 평균 계산

calculate_average(score_list) 
def calculate_average(score_list):
    total = 0

    for score in score_list:   # 점수 목록을 하나씩 순회
        total += score           # 누적 합산

    return total / len(score_list)  # 평균 = 합계 ÷ 개수
✅
len(score_list)는 리스트의 길이(원소 개수)를 반환합니다. 점수가 3개이면 len([88, 92, 81]) → 3 이므로 총합을 3으로 나눕니다.

② make_level – 등급 판정

make_level(avg) 
def make_level(avg):
    if avg >= 90:
        return "A"
    elif avg >= 80:
        return "B"
    elif avg >= 70:
        return "C"
    else:
        return "D"
조건등급예시 평균
avg ≥ 90A95.0 → A
80 ≤ avg < 90B87.0 → B
70 ≤ avg < 80C72.3 → C
avg < 70D65.0 → D
💡
elif는 앞 조건이 거짓일 때만 검사합니다. 즉 avg가 95.0이면 첫 번째 if avg >= 90이 참이므로 나머지 elif/else는 아예 실행되지 않습니다.

③ print_report – 전체 리포트 출력

print_report(student_list) 
def print_report(student_list):
    print("=== 학생 성적 리포트 ===")

    for student in student_list:       # 학생 한 명씩 순회
        name   = student["name"]
        scores = student["scores"]

        average = calculate_average(scores)  # ① 함수 호출
        level   = make_level(average)           # ② 함수 호출

        if average >= 70:                      # 합격 여부 판정
            result = "합격"
        else:
            result = "보충 필요"

        print(f"이름: {name}")
        print(f"점수 목록: {scores}")
        print(f"평균: {average:.1f}, 등급: {level}, 결과: {result}")
        print("-")
🔍
f-string 포매팅 – {average:.1f} 에서 :.1f는 소수점 첫째 자리까지 표시하라는 형식 지정자입니다. 예: 87.666... → 87.7

5. 실행 흐름 추적

민지(점수: 88, 92, 81)를 예로 한 단계씩 따라가 봅시다.

단계 호출 처리 내용 결과
1 print_report 시작 student = {"name":"민지", "scores":[88,92,81]} –
2 calculate_average([88,92,81]) (88+92+81) ÷ 3 = 261 ÷ 3 87.0
3 make_level(87.0) 87.0 < 90 → elif 87.0 ≥ 80 → True "B"
4 합격 여부 87.0 ≥ 70 → True "합격"
5 출력 f"평균: 87.0, 등급: B, 결과: 합격" 화면 출력
