카페 주문 계산기

딕셔너리 조회, 다중 반환값, 함수 호출 체인을 익히는 종합 예제

list dict for 반복문 if / else 함수 (def) 다중 반환값 in 연산자 f-string

1. 학습 목표

🔍
딕셔너리 조회
in 연산자로 키 존재 여부를 확인하고 값을 가져온다.
↩️
다중 반환값
함수가 값을 여러 개 돌려주고, 호출 측에서 분리해 받는 방법을 이해한다.
🔗
함수 호출 체인
함수의 결과를 다른 함수에 넘겨 단계적으로 처리하는 흐름을 익힌다.
📝
예외 데이터 처리
메뉴에 없는 항목을 별도 리스트에 모아 사용자에게 알려주는 패턴을 익힌다.

2. 전체 프로그램 구조

이 예제는 데이터 2개와 함수 3개로 이루어져 있습니다.

D

4. 함수별 코드 해설

① calculate_total – 합계 계산 + 미등록 메뉴 수집

calculate_total(order_list, price_table) 
def calculate_total(order_list, price_table):
    total = 0
    unknown_items = []              # 미등록 메뉴 담을 빈 리스트

    for item in order_list:
        if item in price_table:       # 딕셔너리 키 존재 여부 확인
            total += price_table[item]  # 가격을 더함
        else:
            unknown_items.append(item)  # 없으면 미등록 목록에 추가

    return total, unknown_items         # 두 값을 함께 반환
💡
item in price_table
딕셔너리에 in을 사용하면 키(key)가 존재하는지 확인합니다. "아메리카노" in menu → True, "주스" in menu → False
✅
다중 반환값 – return total, unknown_items
Python은 쉼표로 구분하면 여러 값을 튜플로 묶어 반환합니다. 받는 쪽에서 total, unknown_items = calculate_total(...)처럼 언패킹해서 사용합니다.

② apply_discount – 할인 적용

apply_discount(total, order_count) 
def apply_discount(total, order_count):
    # 4개 이상 주문 시 10% 할인
    if order_count >= 4:
        discounted = int(total * 0.9)    # 90% = 10% 할인
        return discounted, "10% 할인 적용"

    return total, "할인 없음"
🔍
int(total * 0.9)
* 0.9는 10% 할인(원가의 90%)을 뜻합니다. 곱셈 결과가 소수가 될 수 있으므로 int()로 정수(원 단위)로 내림 처리합니다. 예: int(14000 * 0.9) → int(12600.0) → 12600
주문 수할인반환 메시지
1 ~ 3개없음"할인 없음"
4개 이상10%"10% 할인 적용"

③ print_bill – 최종 영수증 출력

print_bill(order_list, price_table) 
def print_bill(order_list, price_table):
    print("=== 카페 주문 계산기 ===")
    print("주문 목록:", order_list)

    # ① calculate_total 호출 → 두 값 언패킹
    total, unknown_items = calculate_total(order_list, price_table)

    # ② apply_discount 호출 → 두 값 언패킹
    final_total, message = apply_discount(total, len(order_list))

    print(f"할인 전 금액: {total}원")
    print(f"최종 금액: {final_total}원 ({message})")

    if unknown_items:                           # 리스트가 비어있지 않으면 True
        print("가격표에 없는 메뉴:", unknown_items)
    else:
        print("모든 메뉴가 정상 계산되었습니다.")
💡
if unknown_items:
빈 리스트 []는 Python에서 False로 취급됩니다. 즉 미등록 메뉴가 없으면 else 블록이 실행되고, 하나라도 있으면 if 블록이 실행됩니다.

5. 다중 반환값 & 언패킹 심화

이 예제의 핵심 개념을 별도로 정리합니다.

다중 반환값 원리 
# 반환 시 튜플로 묶임
return total, unknown_items
# 위는 사실 아래와 같습니다
return (total, unknown_items)

# 받는 쪽에서 언패킹 (튜플을 각 변수에 분리)
total, unknown_items = calculate_total(order_list, price_table)
# 위는 사실 아래와 같습니다
result = calculate_total(order_list, price_table)
total          = result[0]
unknown_items  = result[1]
✅
언패킹을 사용하면 반환값을 바로 의미있는 이름의 변수에 담을 수 있어 코드 가독성이 높아집니다. 왼쪽 변수 개수와 반환값 개수가 반드시 같아야 합니다.

8. 핵심 개념 정리

리스트의 in과 같은 문법이지만 딕셔너리는 키만 검색.
다중 반환 & 언패킹
return a, b → 튜플 반환
x, y = func() → 각 변수에 분리해서 받기
append 로 리스트 누적
빈 리스트 []를 만들고 조건에 맞는 항목만 .append()로 추가하는 패턴.
if 리스트: 빈 값 체크

원소가 있으면 True 로 조건 판단 가능.
