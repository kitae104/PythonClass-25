# 숫자 맞추기 게임 응용 문제

기본 숫자 맞추기 게임을 응용하여 아래 문제들을 해결해 보세요.  
각 문제는 독립적으로 작성할 수 있으며, 원하는 문제를 선택하거나 여러 개를 풀어도 됩니다.  

---

## 📌 문제 1. 범위 확장 버전
기존 숫자 맞추기 게임은 1~100 사이에서 진행됩니다.  
이번에는 **사용자가 직접 범위를 입력**해서 (예: 1~500, 10~50) 게임이 진행되도록 프로그램을 수정하세요.  

- 사용자에게 “시작 숫자”와 “끝 숫자”를 입력받는다.  
- 그 범위 내에서 정답을 랜덤으로 선택한다.  
- 나머지 진행은 기본 게임과 동일하다.  

---

## 📌 문제 2. 제한된 기회 버전
사용자가 무한정 시도할 수 있던 게임을 **기회 제한**이 있는 게임으로 수정하세요.  

- 기회는 7번으로 제한한다.  
- 시도 횟수를 모두 소진하면 `"실패! 정답은 OOO"` 이라고 출력한다.  
- 정답을 맞추면 바로 게임이 끝난다.  

---

## 📌 문제 3. 두 명이 겨루는 대전 모드
두 명의 플레이어가 번갈아 가며 숫자를 입력하여 누가 먼저 맞추는지 겨루는 프로그램을 작성하세요.  

- Player 1, Player 2 입력을 번갈아 받는다.  
- 정답을 맞춘 사람이 승리한다.  
- 시도 횟수와 승자를 출력한다.  

---

## 📌 문제 4. 난이도 조절 버전
게임 시작 시 난이도를 선택할 수 있게 하세요.  

- **쉬움 모드:** 숫자 범위 1~50  
- **보통 모드:** 숫자 범위 1~100  
- **어려움 모드:** 숫자 범위 1~1000  

사용자가 난이도를 선택하면 해당 범위에서 정답을 고르고 게임을 시작합니다.  

---

## 📌 문제 5. 힌트 제공 버전
숫자를 입력했을 때 단순히 “업/다운”만 알려주는 것이 아니라, **추가 힌트**를 제공하도록 만드세요.  

예시:  
- `"업! 그리고 정답은 짝수예요."`  
- `"다운! 정답은 50보다 작습니다."`
