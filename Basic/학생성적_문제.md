# 과제: 학생 성적 리포트 프로그램 작성

학습 목표

- 리스트, 딕셔너리 사용법 숙달
- 반복문과 조건문을 사용한 자료 처리
- 함수를 정의하여 코드 구조화하기
- 문자열 포매팅(f-string)으로 출력 형식 맞추기

과제 설명

- 아래에 주어진 학생 데이터(리스트)를 이용해 각 학생의 평균과 등급을 계산하고, 콘솔에 보기 쉬운 성적 리포트를 출력하는 프로그램을 작성하세요.
- 구현해야 할 핵심 함수는 `calculate_average`, `make_level`, `print_report` 입니다.

데이터 (문제에 포함할 초기 데이터)

```python
students = [
    {"name": "민지", "scores": [88, 92, 81]},
    {"name": "준호", "scores": [70, 65, 72]},
    {"name": "서연", "scores": [95, 90, 98]},
]
```

요구사항

1. `calculate_average(score_list)`
    - `score_list`에 들어있는 숫자들의 평균을 계산하여 실수(float)로 반환하세요.
    - 소수점 표현은 출력 단계에서 `:.1f`로 포맷합니다.
2. `make_level(avg)`
    - 평균 점수 `avg`를 받아 등급을 반환하세요.
    - 등급 기준: A: >=90, B: >=80, C: >=70, D: <70
3. `print_report(student_list)`
    - 다음 형식으로 리포트를 출력하세요.
        - 첫 줄은 `=== 학생 성적 리포트 ===`
        - 학생마다 이름, 점수 목록, 평균(소수점 1자리), 등급, 결과(합격/보충 필요)를 출력
        - 평균이 70 이상이면 `합격`, 미만이면 `보충 필요`로 표시
    - 각 학생 출력 뒤에는 `-` 한 줄을 넣으세요.
4. 프로그램 진입점에서 `print_report(students)`를 호출하면 위 출력이 나오도록 하세요.
