# 과제: 카페 주문 계산기 프로그램 작성

학습 목표
- 리스트와 딕셔너리 사용에 익숙해지기
- 반복문과 조건문을 사용한 자료 처리 능력 강화
- 함수를 정의하여 프로그램 구조화하기
- 예외(미등록 메뉴) 처리 연습

과제 설명
- 아래에 제공된 `menu`(가격표)와 `orders`(주문 목록)를 사용해 주문 금액을 계산하고, 할인 규칙을 적용하여 최종 금액을 출력하는 프로그램을 작성하세요.
- 미등록 메뉴(가격표에 없는 항목)는 무시하되, 사용자에게 알려주어야 합니다.

데이터 (문제에 포함할 초기 데이터)
```python
menu = {
    "아메리카노": 3000,
    "카페라떼": 4000,
    "쿠키": 2000,
    "샌드위치": 5000,
}

orders = ["아메리카노", "쿠키", "카페라떼", "샌드위치", "주스"]
```

요구사항
1. `calculate_total(order_list, price_table)` 구현
   - `order_list`의 항목들을 순회하며 `price_table`에 있는 항목만 합산하여 정수 합계를 반환하세요.
   - 가격표에 없는 항목은 수집하여 리스트로 반환하세요. (반환값 예: `(total, unknown_items)`)
   - 미등록 항목이 있으면 `print`로 경고 메시지를 출력해도 됩니다.
2. `apply_discount(total, order_count)` 구현
   - 주문 개수가 4개 이상이면 총액의 10%를 할인합니다.
   - 할인 적용 후의 금액(정수)과 할인 메시지(예: "10% 할인 적용" 또는 "할인 없음")를 반환하세요.
3. `print_bill(order_list, price_table)` 구현
   - 출력 헤더로 `=== 카페 주문 계산기 ===`를 출력합니다.
   - 주문 목록을 출력하고, 할인 전 금액, 최종 금액(할인 메시지 포함), 그리고 가격표에 없는 메뉴 리스트를 출력하세요.
   - (선택) 계산된 영수증을 `bill.txt`로 저장하는 기능을 추가해도 됩니다.
4. 단, 프로그램이 빈 주문 리스트에도 안전하게 동작하도록 구현하세요.

예시 출력 (제공된 데이터로 실행했을 때)
```
=== 카페 주문 계산기 ===
주문 목록: ['아메리카노', '쿠키', '카페라떼', '샌드위치', '주스']
할인 전 금액: 14000원
최종 금액: 12600원 (10% 할인 적용)
가격표에 없는 메뉴: ['주스']
```
